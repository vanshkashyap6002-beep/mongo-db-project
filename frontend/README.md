# 📋 MERN Todo App - Frontend

A simple and responsive Todo List application built with **React** and **Vite**. This frontend communicates with a Node.js + Express backend using REST APIs and allows users to manage their daily tasks efficiently.

---

## 🚀 Features

* ➕ Add new tasks
* 📝 Edit existing tasks
* 🗑️ Delete tasks
* ✅ Update task status
* 🔍 Search tasks
* ⏳ Loading state while making API requests
* ⚠️ Basic error handling
* 📱 Responsive user interface
* 🌐 API integration using Axios

---

## 🛠️ Tech Stack

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Go to the frontend folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
VITE_API_URL=YOUR_BACKEND_API_URL
```

Example:

```env
VITE_API_URL=https://your-backend-url/api/tasks
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
